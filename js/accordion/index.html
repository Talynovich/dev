Секция 1: О DOM

DOM – это программный интерфейс для HTML-документов. Он позволяет JS взаимодействовать со страницей.

Секция 2: События

Для обработки кликов используется addEventListener. События всплывают по дереву DOM.

Секция 3: Делегирование

Один обработчик на родителе вместо множества обработчиков на детях, что экономит память.
